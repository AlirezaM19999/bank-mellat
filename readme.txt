== Description ==
افزونه پرداخت آنلاین بانک ملت

توسط این افزونه می توانید با استفاده از یک کد کوتاه، امکان پرداخت آنلاین بانک ملت را به وردپرس خود اضافه نمائید.

امکانات:

* دروازه پرداخت بانک ملت
* ارسال ایمیل بعد از پرداخت ها
* ارسال پیامک بعد از هر پرداخت
* 3 فرم مختلف
* امکان ایجاد فرم های سفارشی
* بخش راهنما

== Installation ==
1. Upload `Bank-mellat` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. go to the plugin setting and enter terminal id,terminal user and terminal password
4. then add shortcode [WPBEGPAY_SC] to one of your posts and pages.
